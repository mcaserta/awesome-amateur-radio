# Contributing to Awesome Amateur Radio

Thank you for your interest in contributing to Awesome Amateur Radio! This guide will help you contribute effectively.

## How to Contribute

### Adding a New Resource

1. **Fork the repository** and create a new branch for your changes
2. **Add your resource** to the appropriate section in `README.md`
3. **Follow the formatting guidelines** below
4. **Submit a pull request** with a clear description of what you're adding

### Formatting Guidelines

- Use the following format for entries: `[Resource Name](URL) - Brief description (optional)`
- For entries without descriptions: `[Resource Name](URL)`
- Keep descriptions concise (under 100 characters when possible)
- Ensure URLs are valid and actively maintained
- Sort entries alphabetically within their section when possible

### Quality Standards

Resources should be:
- **Relevant** to amateur radio operators
- **Active** and maintained (not abandoned projects)
- **High quality** and valuable to the community
- **Legal** and properly licensed
- **Accessible** (preferably free or with clear pricing)

### What to Include

Great additions include:
- Open source software and tools
- Educational resources and tutorials
- Active communities and forums
- Equipment guides and reviews
- Technical references and documentation
- Online services beneficial to ham radio operators

### What NOT to Include

Please avoid:
- Paid services without clear value proposition
- Duplicate resources already listed
- Commercial spam or self-promotion without community value
- Illegal or unlicensed content
- Abandoned or unmaintained projects

### Updating Existing Entries

If you find:
- **Broken links**: Submit a PR with a fix or replacement
- **Outdated information**: Update with current details
- **Better descriptions**: Improve clarity while staying concise

### Section Organization

The repository is organized by topic:
- **Antennas and Propagation** - Antenna designs, analyzers, propagation tools
- **Equipment (Rigs)** - Radios and accessories
- **Digital Modes** - FT8, WSJT-X, packet radio, etc.
- **Morse Code (CW)** - Learning resources and tools
- **Satellites** - Satellite operation and tracking
- **SDR** - Software Defined Radio resources
- **Organizations and Regulation** - Ham radio organizations and band plans
- **Software** - Applications for logging, control, and planning
- **Learning and Training** - Educational resources
- **Shopping** - Equipment vendors
- **Regional Resources** - Country-specific information

If your resource doesn't fit existing sections, suggest a new section in your PR.

### Adding Manuals or Assets

For equipment manuals or other binary files:
- Only include **publicly available** manuals (not copyrighted material)
- Keep file sizes reasonable (under 10MB when possible)
- Name files descriptively: `manufacturer-model.pdf`
- Place manuals in `/manuals/` directory
- Place other assets in `/assets/` directory
- Reference them in README.md with relative paths

### Pull Request Process

1. **Check for duplicates**: Search existing issues and PRs
2. **One topic per PR**: Don't mix unrelated changes
3. **Clear title**: Describe what you're adding (e.g., "Add FreeDV digital voice software")
4. **Description**: Explain why this resource is valuable
5. **Test your links**: Verify all URLs work before submitting
6. **Be patient**: Maintainers will review when possible

### Code of Conduct

Please be respectful and constructive in all interactions. We're here to help each other enjoy amateur radio!

### Questions?

If you're unsure about anything:
- Open an issue to discuss before submitting a PR
- Check the [awesome-list manifesto](https://github.com/sindresorhus/awesome/blob/main/awesome.md) for general guidelines
- Review existing entries for formatting examples

## Thank You!

Your contributions help make this resource valuable for the entire amateur radio community. 73!
