# Digital Modes

Resources for digital communication modes including FT8, WSJT-X, packet radio, FreeDV, and more.

## Software and Protocols

- [FreeDV](https://freedv.org/) - Open source digital voice for HF radio
- [FT4 Protocol](https://physics.princeton.edu//pulsar/k1jt/FT4_Protocol.pdf) - Technical documentation
- [The FT4 and FT8 Communication Protocols](https://physics.princeton.edu/pulsar/k1jt/FT4_FT8_QEX.pdf) - Detailed protocol specifications
- [FT8 QRG](https://www.sigidwiki.com/wiki/FT8#Frequencies) - FT8 frequency guide
- [FT8 intro](https://docs.google.com/presentation/d/e/2PACX-1vRDYMNACPkwAs5tVkFfLibA08quk3ClhhzzI8N0WUJxkzdNOCEnd50yqvAcKbACyJwx-r5ZTH6S7YPY/pub) - Introduction presentation
- [FT8 tips](https://www.g4ifb.com/FT8_Hinson_tips_for_HF_DXers.pdf) - Tips for HF DXers
- [Gridtracker](https://gridtracker.org/grid-tracker/) - Companion application for WSJT-X and JTDX
- [JTDX](https://www.jtdx.tech/en/) - Improved version of WSJT-X | [source](https://github.com/jtdx-project/jtdx)
- [M17 Project](https://m17project.org/) - Open source digital radio protocol
- [PSK Reporter](https://pskreporter.info/) - Reception reporting system
- [Q65 Quick Start](https://physics.princeton.edu/pulsar/k1jt/Q65_Quick_Start.pdf) - Guide to Q65 mode
- [WSJT-X](https://physics.princeton.edu/pulsar/k1jt/wsjtx.html) - Digital modes software suite
- [WSJT-X User Guide](https://physics.princeton.edu/pulsar/k1jt/wsjtx-doc/wsjtx-main-2.4.0.html) - Comprehensive documentation
- [WSJT-X Split Operation](https://www.k0pir.us/wsjt-x-split-operation/) - Guide for split frequency operation
- [WSPRnet](http://wsprnet.org/) - Weak Signal Propagation Reporter Network

## Packet Radio

- [ISS Packet Operation](https://www.ariss.org/uploads/1/9/6/8/19681527/k9jkm_2012_symposium_ver2.pdf) - Operating packet via ISS
- [Packet Radio: LinBPQ Ready to Go](https://eindhoven.space/2020/11/01/linux-linbpq-ready-to-go/) - Linux packet radio setup
- [QTSoundModem](https://www.cantab.net/users/john.wiseman/Documents/QtSoundModem.html) - Soundcard based packet modem
- [Sound Card Packet](https://www.soundcardpacket.org/) - Resources for soundcard packet radio

## Messaging and Gateways

- [SmsGte](https://smsgte.org/) - SMS gateway for amateur radio
